最近我发现了一道十分困难de题，内容大概是这样


---
输入若干组数，输出每一组数的和

输入样例1：
```
1 2
3 4
5 6
```
输出样例1：
```
3
7
11
```

输入样例2：
```
1 3
2 4
```
输出样例2：
```
4
6
```
---

看完这题我当场就EMO了，它没告诉我有几组数据啊！！！

于是我凑活着写了一段代码：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int RE,WA;
    while (true){
        cin >> RE >> WA;
        cout << RE + WA << '\n';
    }
    return 0;
}
```
于是喜提 --> `TLE`

```
TLE , Time Limit Error

指代码没有在规定时间内结束运行
```

最后,我还是知道了这题是怎么做的了(别问)

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int WA,RE;
    while (cin >> WA >> RE){
        cout << WA + RE << '\n';
    }
    return 0;
}
```
就是这么神奇......

于是我开思考cin有没有返回值......代码如下
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int a;
    cout << cin >> a;
    return 0;
}
```
